Undergraduate Research

Dynamics and Control Systems Laboratory

I started as an Undergraduate Research Assistant in the Dynamics and Control Systems Laboratory at Georgia Tech in Fall of 2024. DCSL is focused on testing Robotic Algorithms and Control Systems in a simulated space like environment.

With DCSL I have worked on the integration of electronic systems into the new COSMOS2 robot including:

  • Integration of a new digital IMU with UART communication
  • Debugging power converters to supply power from a battery to a Raspberry Pi
  • Integration and firmware for a board to control solenoid thrusters with PWM
  • Troubleshooting and testing electronic subsystems in simulated environments

I have also taken on a number of Control Systems and Robotic Software related projects including:

  • Integration of the new IMU into the existing Kalman filter
  • Integration of a SLAM pipeline into the ROS2 framework
  • Use of signal processing, regression and rotational dynamics to experimentally derive IMU frame transformations

Class Highlights

Control System Design (ECE 4550)

Design and analysis of control systems using state space methods. Lab component focused on implementing control algorithms on a micro controller. Culminated in the stabilization of an inverted pendulum.

Feedback Control Systems (ECE 3550)

Fundamentals of classical feedback control including stability analysis with root locus plots, Nyquist plots, and Bode plots.

Digital Signal Processing (ECE 4270)

Digital filter theory and implementation using Fourier Transforms, Z Transforms. Introduction to random signal processing and statistical analysis.

Electromagnetic Applications (ECE 4350)

2nd course on electromagnetic physics using maxwells equations. Focus on applications to RF systems.

Machine Learning (CS 4641)

Introduction to statistical techniques, machine learning techniques and numerical optimization.

Special Problems (ECE 4902)

Fundamentals of optimal control with a focus on exploring sample problems and underlying mathematics.

Integrated Optics (ECE 6520) In Progress

Electromagnetic theory applied to optical systems.

Random Signals and Applications (ECE 4260) In Progress

Advanced statistical techniques and applications to signal processing.

Automation and Robotics (ECE 4560) In Progress
